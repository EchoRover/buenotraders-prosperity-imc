# Project Documentation — Evan's Workspace

This folder contains chronological documentation for Evan's work in the **buenotraders** team for the **IMC Prosperity** trading competition.

## Structure

| File | Purpose |
|------|---------|
| `project-overview.md` | High-level goals, team info, competition details |
| `strategy-log.md` | Trading strategies — ideas, implementations, results |
| `technical-log.md` | Code architecture, algorithms, technical decisions |
| `research-notes.md` | Market analysis, data exploration, findings |
| `changelog.md` | Chronological record of all significant changes |

## Conventions

- Every entry is dated with `## YYYY-MM-DD` headers
- Newest entries go at the top of each file
- Files are continuously updated — they grow over time
- Keep entries concise but detailed enough to be useful months later
