# Logs

Drop competition log zips here. Naming convention:

**From competition:** Just drop the zip as-is. Files will be extracted and renamed to:
`{model}_{submission_id}.json` / `.py` / `.log`

**Processed logs are named:**
- `e1_v1_final_45769.json` — e1_v1 final version, submission ID 45769
- `arjun_45684.json` — teammate algo, submission ID 45684

Each `.json` contains: `profit`, `activitiesLog`, `graphLog`, `positions`
Each `.py` contains: the exact code that was submitted
